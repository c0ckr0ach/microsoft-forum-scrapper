Installing Windows 11 fails with multiple BSOD error screens despite PC passing PC Health Check for Windows 11

Nathan Petty 0 Reputation points
2025-09-09T16:02:17.45+00:00

I have a custom built PC currently running Windows 10 that I would like to run Windows 11. The hardware is as follows:

  • Intel i5-9600k
  • Gigabyte B365M DS3H on F7 firmware
  • Crucial 8GB DDR4 2666
  • WD Blue 2.5' SATAIII 256GB SSD

I've enabled Intel PTT and Windows shows the TPM 2.0 active. CSM is disabled so that UEFI is being used and Secure Boot is enabled. I've tried doing an in place upgrade, running setup from a Windows 11 ISO mounted locally as well as running it from a USB and cleaning the disk first. Every option fails with various BSOD screens. In particular, I've seen "Kernel Security Check Failure", "IRQL_NOT_LESS_THAN_OR_EQUAL", "MEMORY_MANAGEMENT" and "WORKER INVALID". It always fails during install, even on a blank drive, before ever getting to the OS. Windows 10 installs without issue, even after the Windows 11 blue screens (I reinstalled Windows 10 to get it back up for now). I've used memtest to check the memory with no errors, swapped out the SSD and changed which slot my RAM is in with no luck. With no issues for Windows 10, it feels like a compatibility issue, but it meets all the Windows 11 requirements on the website and passed the PC health check without issue. Has anyone seen this issue before? Thanks!
